02. 毕业必须通过的5个项目

المشروعات الفصل الثاني

当你从纳米学位毕业的时候，你将掌握相关技能，并学会独立工作。我们希望尽快帮你实现目标，这也是为什么我们注重从 实践项目 中学习的原因。

### 在数据分析（进阶）纳米学位中，你需要完成下面这些课程和内容：

  • 第 1 部分：用 Python 进行简单数据分析（可试学）

复习和巩固如何使用 Python 进行数据分析的基本流程。

  • 实战项目：揭秘《冰与火之歌》战役制胜因素（可试学）

获得《权力的游戏》五大家族的战争数据，分析影响战争走势的因素，温习从提问到可视化结论的数据分析流程，并找到隐藏在文字中的战役制胜法宝。

  • 第 2 部分：使用统计学验证真实应用中的假设

复习巩固概率论、贝叶斯定理、描述统计学、假设检验基础，以及运用 Python 编写函数和调用 Python 库的能力。

  • 实战项目 2：验证经典心理学现象

使用描述性统计和假设检验，分析经典心理学实验“斯特鲁普效应”数据，提出你自己的假设，阐述你对数据的理解，并通过统计推论得出结论。

  • 第 3 部分：数据清洗

深入掌握如何从本地文件、API、互联网等不同来源收集数据，并进行有效清洗；使用 Pandas 评估数据，最后使用 Python 清洗和校验。

  • 实战项目 3：清洗和分析 5000 条狗狗评分 Twitter

你将获得超过 5000 条 Twitter 上人们对狗狗照片的评分数据，它们很“脏”，你需要用学到的技能，评估数据质量和整洁度，并进行清洗、分析和可视化，最终得出属于你自己的有趣结论！

  • 第 4 部分：探索性数据分析

了解什么是探索性数据分析，并从配置环境开始，逐步掌握用 R 实现探索性数据分析和可视化。最后你将与 Facebook 数据科学家 Solomon Messing 一起进行钻石价格分析实战，巩固学到的内容。

  • 实战项目 4：探索性分析葡萄酒品质、总统大选募资数据

使用 R 分析“什么物质在影响葡萄酒品质评分？”“总统候选人性别在如何影响他们获得的财政捐助”等真实问题，并得出属于你自己的结论。

  • 第 5 部分：数据可视化：用数据来讲故事

深入了解数据可视化的重要性，包括不同的数据类型编码方式：色彩、形状、大小等；熟练掌握 Tableau 基本功能，并学习创建仪表盘和故事，实现生动有力的数据可视化。

  • 实战项目 5：可视化摩拜单车用户行为中的数据故事

运用 Tableau，可视化摩拜单车上海用户行为数据中的故事和趋势。你将熟悉可视化理论和应用，还将学会如何利用视觉编码和美学参数实现令人难忘的沟通。还有更多真实前沿的数据集等你探索。

你可以通过学习课程为实战项目做准备，但学完所有课程材料 并不是 毕业的必要条件， 完成课程中涉及的五个实战项目是毕业的必要条件 。如果你在上课前已经拥有丰富的经验，或有更适合你的材料，请你试着 直接 投入到实战项目中。然而大多数学生需要学习课程内容，并完成练习，从而能在项目中获得成功。

为帮助你快速实现目标，我们为你的每个实战项目设置了建议完成日期，显示在教室中课程主页相关项目旁。此时间线的设计标准为每周大约投入 10 小时。 请在任何可能的时候回到教室继续学习 ，即使只是 10 分钟，也能确保不断取得进步。
