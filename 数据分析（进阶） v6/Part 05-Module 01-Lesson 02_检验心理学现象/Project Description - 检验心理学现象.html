检验心理学现象

Compute descriptive statistics and perform a statistical test on a data set based on a psychological phenomenon.

使用描述统计学和统计检验分析斯特鲁普效应——一个实验心理学的经典成果。为读者提供直观的数据，并根据结果利用统计推断得出结论。

评估

获得可以回答之前页面上问题的文档后，请务必在提交报告前核对项目 评估准则 。只有在你的项目符合所有条件中的规格要求时，才会被评估为“符合规格要求”。如果存在使你不符合规格要求的类别，请务必修改报告，使其符合量规条件。

提交

准备好提交你的项目了吗？请准备好以下文件：

  • pdf 或 html 文档，包含能够回答之前页面上问题的答案。
  • 你在此次提交过程中参考或使用的网站、书籍、论坛、GitHub 库和其他资源（网络资源或非网络资源）的列表。该列表可以随附在答案文档的底部，或者放在单独的纯文本文件中。

点击页面下方的“提交项目”，然后根据说明提交：

  • 如果你要通过“项目链接”提交文件，请将项目文件上传到 GitHub，然后将链接发送给我们。
  • 如果你要通过“上传文档”提交文件，请压缩项目目录，然后提交 zip 文件。在上传之前，我们建议将 zip 文件以及其内部文件的文件名改为 英文 ，以尽可能避免上传过程中的问题发生。

我们最多需要一周的时间来对项目进行评分，因此请随时检查是否有更新。

如果你在项目提交上有任何问题，或者你希望查看提交状态，请发送电子邮件至 support@youdaxue.com

提醒：如果你选择上传文档，你需要用英文命名文档，文件名中若有中文字符，会导致提交失败。
