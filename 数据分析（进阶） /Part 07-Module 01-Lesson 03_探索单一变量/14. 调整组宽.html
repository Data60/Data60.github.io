你会在下方代码中添加什么代码，以按性别构建好友数直方图分面？
在你本地电脑上的 RStudio 中创建分面的直方图。你是否成功创建了此图？

ggplot2 中的标尺




等效的 ggplot 语法：


ggplot(aes(x = friend_count), data = pf) +


geom_histogram(binwidth = 25) +


scale_x_continuous(limits = c(0, 1000), breaks = seq(0, 1000, 50))




qplot(x = friend_count, data = pf, binwidth = 25) +


scale_x_continuous(limits = c(0, 1000), breaks = seq(0, 1000, 50))

在以下的可选解决方案的 facet_grid() 公式中，句点代表数据集中的其他所有变量，这一符号实质上根据性别拆分数据，并且生成三个直方图，每个图都有自己的行。


qplot(x = friend_count, data = pf) +


facet_grid(. ~ gender)




等效的 ggplot 语法：


ggplot(aes(x = friend_count), data = pf) +


geom_histogram() +


scale_x_continuous(limits = c(0, 1000), breaks = seq(0, 1000, 50)) +


facet_wrap(~gender)
