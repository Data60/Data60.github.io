在 {r} 代码块的内部和外部使用 “pound sign/key”或称 "hash sign/key" # 会有所不同。 

```{r}
# r 块中的 # (pound / hash sign) 会创建
# 注释。这三行不是代码，因此不能被 # 执行。
x <- [1:10]
mean(x)
```

如果你在 {r} 代码块外部使用 # 符号，你可以创建文本头。
