03. 项目模板 RMD 文件

项目模板 RMD 文件

请下载 项目模板文件 ，或从下载 EDA 课程教材压缩文件 中找到该模版文件，开始你的分析。

格式注意事项

我们希望你提交一个可读的 RMD 文件。为了帮助你准备项目，请查看以下注意事项。

  1. 拼合的 HTML 输出应可读。请务必查看你拼合的 HTML 文件，并且检查代码和图形是否正确。

  2. 使用哈希或英镑符号将 RMD 或 R-Markdown 文件中 R 代码的注释包含在 r 代码块中。 

    3. ```{r}
library(ggplot2)
# 这是一个注释示例，不是实际的代码。
```

  3. 在 RMD 或 R-Markdown 文档中，在 r 代码块外加上 `#` 符号将创建一个 H1 标题。

    4. 这是一个 H1 标题

    导出为 HTML 格式时，`#` 将会变成不可见。 点击这里 查看 Markdown 语法说明。

  4. 请确认所有图片在导出的 HTML 文件中显示正常。
