06. 宽和长格式

宽和长格式

INSTRUCTOR NOTE:

你还可以使用 tidyr 包重构数据。你可以在 PDF 版本的 使用 R 进行的数据整理 中查看示例，了解包的使用方法。

以下为将数据框从长格式变为宽格式（整洁格式）的代码。我们鼓励你在查看以下解决方案之前，阅读 PDF 版本的“数据整理”，并且使用 tidyr 包编写代码。







install.packages("tidyr") # only need to run this once
library(tidyr)
spread(subset(pf.fc_by_age_gender, select = c('gender', 'age', 'median_friend_count')), gender, median_friend_count)


tidyr 包和 reshape2 包都可以完成此任务，但用 tidyr 可能会容易一点。
